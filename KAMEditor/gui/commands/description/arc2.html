Быстрая клавиша Shift+9
ОТН. ДУГА [ Плоскость, ∆X, ∆Y,R, (СКОРОСТЬ)]
Перемещение на [∆X, ∆Y, R] по дуге радиуса R в выбранной плоскости.
∆X – Перемещение по X.
∆Y – Перемещение по Y.
R – Радиус дуги.
Если R со знаком "-", то движение по дуге против часовой стрелки
(СКОРОСТЬ) Скорость выполнения оператора (необязательный параметр).
